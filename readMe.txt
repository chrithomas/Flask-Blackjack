personal project to help me learn python-based web development with flask

STARTUP
---------------------------
app % python3 -m venv .venv
[MAC ONLY] app% . .venv/bin/activate 
[WIN ONLY] app> .venv/Scripts/activate
(.venv) app % pip install flask
(.venv) app % flask --app app run --debug
